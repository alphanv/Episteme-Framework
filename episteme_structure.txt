### Custom Environment (`examples/custom_environment.py`)

```python
"""Example: Creating a custom environment"""

from episteme.environments import BaseEnvironment
import numpy as np

class CustomEnvironment(BaseEnvironment):
    """Your custom environment implementation"""
    
    def __init__(self, **kwargs):
        super().__init__(**kwargs)
        self.state = self.initial_state()
    
    def step(self, action):
        """Execute action and return observation"""
        # Your dynamics here
        self.state = self.dynamics(self.state, action)
        return self.observe(self.state)
    
    def dynamics(self, state, action):
        """Define state transition"""
        # Implement your dynamics
        return state
    
    def observe(self, state):
        """Generate observation with noise"""
        return state + np.random.normal(0, self.obs_noise)
```

## ðŸš€ Deployment & CI/CD

### `.github/workflows/tests.yml`

```yaml
name: Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        python-version: [3.8, 3.9, '3.10']
    
    steps:
    - uses: actions/checkout@v2
    - name: Set up Python ${{ matrix.python-version }}
      uses: actions/setup-python@v2
      with:
        python-version: ${{ matrix.python-version }}
    
    - name: Install dependencies
      run: |
        python -m pip install --upgrade pip
        pip install -r requirements.txt
        pip install -r requirements-dev.txt
    
    - name: Run tests
      run: pytest --cov=episteme tests/
    
    - name: Check formatting
      run: |
        black --check episteme/ tests/
        flake8 episteme/ tests/
```
